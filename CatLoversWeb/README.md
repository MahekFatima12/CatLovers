# 🐱 CatLoversWeb

CatLoversWeb is an ASP.NET Core MVC web application built to practice real-world MVC architecture, Entity Framework Core, and full-stack development concepts.

---

## 🚀 Purpose of This Project

This project was created to:

- Strengthen ASP.NET Core MVC fundamentals
- Understand real-world MVC architecture
- Practice Entity Framework Core with SQL Server
- Learn controller–view–model flow
- Improve UI integration with backend logic
- Gain confidence in debugging and routing issues

---

## 🛠️ Technologies Used

- ASP.NET Core MVC (.NET 7 / .NET 8)
- Entity Framework Core
- SQL Server (LocalDB)
- Razor Views
- Bootstrap 5
- C#
- Visual Studio 2022
